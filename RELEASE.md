## üéØ Game Features Overview

### Core Gameplay
- **Spaceship Navigation**: Intuitive left/right touch controls
- **Asteroid Dodging**: Navigate through falling red asteroids
- **Coin Collection**: Collect yellow coins for points
- **Progressive Speed**: Game speed increases with score
- **Lives System**: 1-3 lives configurable in settings

### Scoring System
- **Base Score**: Difficulty multiplier √ó 1 point per coin
- **Magnet Bonus**: +5 points when coin magnet is OFF
- **Examples**:
  - 1x difficulty + magnet ON = 1 point per coin
  - 3x difficulty + magnet OFF = 8 points per coin
  - 6x difficulty + magnet OFF = 11 points per coin

### Difficulty Levels
- **0.5x**: Very Easy - Perfect for beginners
- **1.0x**: Easy - Standard gameplay

- **3.0x**: Hard - High asteroid density
- **4.0x**: Very Hard - Intense gameplay

## üêõ Known Issues & Solutions

### Common Issues

#### Game Won't Start
- **Issue**: App crashes on launch
- **Solution**: Restart device, clear app cache, reinstall

#### Performance Issues
- **Issue**: Lag or low frame rate
- **Solution**: Disable visual effects, close other apps, restart device

#### Touch Not Responding
- **Issue**: Buttons don't respond to touch
- **Solution**: Check screen protector, restart app, clean screen

#### Sound Issues
- **Issue**: No sound effects or music
- **Solution**: Check device volume, enable sound in settings, restart app
